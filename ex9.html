    How to do?


    Делаем сайт более красивым

    Давайте поменям цвет фона на более приятный, например серый, а шрифт сделаем белым у параграфов, и красным у ссылок.

    html {
    background-color: rgb(15, 15, 15);
    }

    a {
    font-size: 30px;
    color: red;
    }

    p {
    font-size: 30px;
    color: white;
    }

    Теперь давайте поменяем нашу страничку: сделаем приветствие, ссылку на нас в соц. сетях, псевдоним.

    <html>
    <head>
    <title>Моя страничка!</title>
    <link rel="stylesheet" href="css/style.css">
    </head>
    <body>
    <p>Привет! Меня зовут dankkaaa, мне 14 лет.</p>
    <a href="https://vk.com/dankkaaa">Мой ВК</a>
    </body>
    </html>

    Вот что вышло:

    Но этот шрифт очень обычен, и не подходит для этого сайта. Для этого давайте зайдём на Google Fonts

    Тут мы видим такой сайт, на нём мы выбераем любой шрифт который вам нравится:

    Я выбрал такой шрифт:


    Нажимаем плюс сверху слева и видем код, его нужно будет вставлять на сайт, в разделе CUSTOMIZE можно поменять шрифт, например выбрать русский язык (cyrillic):


    Теперь вставляем код на наш сайт, первый код, <link href=.... нужно вставить в тэг head, после кода подключения css

    <html>
    <head>
    <title>Моя страничка!</title>
    <link rel="stylesheet" href="css/style.css">
    <link href="https://fonts.googleapis.com/css?family=Open+Sans&display=swap&subset=cyrillic" rel="stylesheet">
    </head>
    <body>
    <p> Hello world! </p>
    </body>
    </html>

    Второй код мы вставляем в css, туда, где мы хотим чтобы испльзовался наш шрифт, если на всём сайте, то вставляем в html, если только в параграфе или другом тэге, вставляем туда. Я буду испльзовать его везде:

    html {
    background-color: rgb(15, 15, 15);
    font-family: 'Open Sans', sans-serif;
    }

    a {
    font-size: 30px;
    color: rgb(255, 0, 0);
    }

    p {
    font-size: 30px;
    color: white;
    }

    Так выглядит намного лучше, можете поэкспериментировать с шрифтами и выбрать любимый для себя
