    Работа с текстом.

    Тут мы будем изучать работу с текстом с помощью тэга <style> для правильности, и для более быстрого изучения css (В этом тэге используются стили css)

    Первый стиль меняет размер шрифта, font-size.

    <html>
    <head>
    <title>Style</title>
    </head>
    <body>
    <p style=":font-size: 23px">
    Теперь этот текст стал 25px
    </p>
    </body>
    </html>

    Следующий стиль будет менять цвет текста, на любой, какой вы захотите, color.

    <html>
    <head>
    <title>Style</title>
    </head>
    <body>
    <p style=":color: red;">
    Теперь этот текст стал красным
    </p>
    <p style=":color: blue;">
    Теперь этот текст стал синим
    </p>
    </body>
    </html>

    Также вы можете менять цвет фона, вы можете сочитать любые стили друг с другом.

    <html>
    <head>
    <title>Style</title>
    </head>
    <body>
    <p style=":color: blue; background-color: yellow;">
    Теперь этот текст стал синим с жёлтым фоном.
    </p>
    </body>
    </html>

    Вернёмся к тэгам, с помощью них мы можем сделать текст полужирным, курсивным, зачёркнутым и т.д.

    <html>
    <head>
    <title>Style</title>
    </head>
    <body>
    <p>
    <b> Полужирный </b>
    <i> Курсив </i>
    <u> Подчёкнутый </u>
    <s> Зачёркнутый </s>
    </p>
    </body>
    </html>
